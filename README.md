# 🏀 Basket Tavolo Ufficiali

App web per la gestione del tavolo ufficiali durante i tornei di basket 3vs3.

## Pagine

- **`index.html`** — Tavolo Ufficiali (organizzatori): punteggio, falli, statistiche, timer, export PDF
- **`public.html`** — Vista Pubblica (tifosi): risultati live e classifica per girone

## Funzionalità

- ⏱ Timer di gioco configurabile
- 🏆 Punteggio in tempo reale con +1/+2/+3
- 🟡 Gestione falli di squadra (con alert bonus)
- 📊 Statistiche giocatori (Pt, Rib, Ast)
- ☁️ Sincronizzazione cloud via Firebase Realtime Database
- 📄 Export PDF del report di giornata
- ✨ Assistente AI integrato (Gemini)
- 📱 Mobile-first e responsive
